# Student-Result-Management-System-Python
Python implementation of a Student Result Management System
It allows managing student records, courses, and exam results in a simple and interactive way. This project is a Python rewrite of a previously SQL-based system.

---

## Features

- Add, update, and delete student records
- Add courses and manage course details
- Record and view student results
- Calculate grades automatically
- Display results in a structured format
- Interactive command-line interface

---

## Technologies Used

- Python 3.x
- Google Colab for development
- File handling for data storage (or optional SQLite)

---

## How to Run

1. Download this repository as a ZIP **or** upload your Python file directly.  
2. Open the Python file (`main.py`) in any Python environment or Colab.  
3. Run the program and follow the menu to:
   - Add students
   - Add courses
   - Record results
   - View student results
